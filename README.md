# Diatonic Keyboard

Play the diatonic button accordion with your computer keyboard!

## Get started

```bash
npm install
npm run dev
```

Navigate to [localhost:8080](http://localhost:8080).

## Building and running in production mode

To create an optimised version of the app:

```bash
npm run build
```
